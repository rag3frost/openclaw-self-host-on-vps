Getting Started

  1. 1

    Pick your AI provider and paste your credentials

  2. 2

    Hook up messaging channels (skip if you want)

  3. 3

    Hit Run Setup and watch the magic

  4. 4

    Click Copy Gateway Token, then open OpenClaw UI

  5. 5
    In the UI, go to Overview → Gateway Token and paste
    i
    OpenClaw no longer accepts tokens via URL for security. You must paste the gateway token in the UI's Overview page to authenticate.
  6. 6
    Come back here → Pair Devices → Approve Latest
    i
    Each new browser needs a one-time device approval. After entering the token and connecting, come back here and approve the pending request.
  7. 7
    Added a bot token? Click Approve Channel Access and drop in the code
    i
    After setup, message your bot on the channel (e.g. send "hey"). It will reply with a pairing code. Enter that here to grant DM access.
  8. 8

    Refresh the page after pairing — you should see Health: OK in the dashboard

Terminal UI (TUI) — You can also access a browser-based terminal by setting ENABLE_WEB_TUI=true in Railway env vars, then navigating to /tui.
