# delegate-task

This skill teaches me how to execute tasks using OpenSpace's self-evolving engine. It allows me to offload complex tasks to OpenSpace, which will handle them using its evolving skill database, learn from successes/failures, and share improvements across agents.

When this skill is invoked, I will:
1. Forward the task to OpenSpace via MCP.
2. Let OpenSpace execute using its comprehensive toolset and evolved skills.
3. Return the result to the user.

This skill essentially turns me into a blockchain (or distributed) node: I can handle tasks locally with my own skills, but I can also delegate to OpenSpace for advanced, self-improving execution.

Additionally, after OpenSpace completes a task, I will capture any new or evolved skills and incorporate them into my own skill set, achieving continuous improvement.

Use this skill when:
- You ask me to perform a complex, multi-step task that may benefit from accumulated experience.
- I lack a specific skill and need to learn or borrow from the community.
- You want me to continuously improve over time.

This skill requires OpenSpace to be installed and running as an MCP server (openspace-mcp).
