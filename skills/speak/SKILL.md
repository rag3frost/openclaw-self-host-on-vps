# speak - Text-to-Speech using ElevenLabs

This skill allows me to convert text into spoken audio using the ElevenLabs API.

## Usage

To use this skill, ensure the `ELEVENLABS_API_KEY` environment variable is set in your OpenClaw environment.

You can then instruct me to speak a particular text. For example:

"Molly, please say: 'Hello, Bjorn! How are you today?'"

## Parameters

Currently, this skill will use default ElevenLabs settings (e.g., a default voice). Future enhancements could include options for different voices, models, or other ElevenLabs parameters.
